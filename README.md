# clickety
Train-themed memory game with React

### Features
* React.js
* Node.js
* Yarn.js
* Adobe Typekit font "Stud"
* Yarn build customized for hosting on a subfolder on my personal website 

### How to play

Try to click on each train, but don't hit any twice!
